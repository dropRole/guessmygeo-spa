## GuessMyGeo SPA

The SPA developed via React framework for GuessMyGeo web app that enables guessing the user's geolocation regarding the informations provided via a users post

## Installation

```bash
$ npm install
```

## Local configuration

### Environment

Create .env file within the root dir, define the PORT var and specify free port on which the app should run on

## Running the app

```bash
# development
$ npm run start

# production-ready
$ npm run build
```
